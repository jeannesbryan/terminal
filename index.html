> TERMINAL UI _

Kerangka kerja CSS/JS ringan tanpa dependensi. Bangun antarmuka web bertema Retro, Cyberpunk, Bunker, atau Sistem Satelit dalam hitungan menit.

BACA DOKUMENTASI FORK REPOSITORY

> SPESIFIKASI_SISTEM

ZERO DEPENDENCY

Tanpa jQuery, tanpa React, tanpa kerumitan. Sepenuhnya dibangun dengan HTML murni, CSS, dan Vanilla JS.

RETRO AESTHETIC

Sudah terintegrasi dengan efek scanlines CRT, skema warna phosphor green, dan kursor berkedip ikonik.

23 MODUL AKTIF

Dilengkapi komponen UI komprehensif mulai dari Sistem Grid, Chat Bubbles, hingga Indikator LED.
