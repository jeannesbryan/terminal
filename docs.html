> MAN_PAGE TERMINAL UI _

Buku panduan lengkap komponen CSS/JS bergaya Bunker/Retro. Terminal UI kini dilengkapi sistem modifikasi tema, validasi form, dan 30 modul antarmuka.

> 01. STARTER TEMPLATE (BOILERPLATE)

Salin kode HTML kosong ini untuk memulai lembaran baru dengan Terminal UI.

> 02. KUSTOMISASI TEMA (CSS VARIABLES)

Terminal UI dirancang untuk dimodifikasi. Anda bisa mengubah skema warna "Matrix Green" menjadi tema "Fallout Amber", "Blood Red", atau "Sci-Fi Cyan" cukup dengan menimpa variabel di file CSS khusus Anda.

/* Tambahkan ini di file CSS custom Anda atau di dalam <style> */
:root {
    /* Tema Fallout (Amber) */
    --t-green: #ffb000;         /* Ganti warna utama menjadi Amber */
    --t-green-dim: #996900;     /* Warna redupnya */
    
    /* Warna latar */
    --bg-base: #110d00;
    --bg-surface: #1f1700;
}

> 04. RESPONSIVITAS & BREAKPOINTS

Terminal UI bersifat Mobile-First. Semua struktur Grid (seperti .t-grid-2, .t-grid-3, dan .t-masonry) akan secara otomatis menjadi 1 kolom penuh apabila diakses di perangkat berlayar kurang dari 768px (Tablet/Mobile). Anda tidak perlu menambahkan kelas CSS responsive ekstra.
